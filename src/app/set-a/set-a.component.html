Q & A

  • 1. How to create reactive forms?

    Answer: To create reactive forms in Angular, import the ReactiveFormsModule from angular/forms in your module. Then, define a FormGroup and FormControl in your component to manage the form's state and validation.

  • 2. How to add built-in validation for email in a reactive form?

    Answer: Use the Validators.email function when creating a FormControl to enforce email format validation.

  • 3. How to add built-in validation for required fields in a reactive form?

    Answer: Apply the Validators.required function to the FormControl to ensure that the field is filled out before submission.

  • 4. How to add regex validation for reactive forms?

    Answer: Use the Validators.pattern function to apply regex validation to a FormControl, allowing for custom validation rules.

  • 5. How to generate routing?

    Answer: Import RouterModule from angular/router and define your routes in the RouterModule.forRoot() method within your module.

  • 6. How to configure routes with parameters?

    Answer: Define a route with a colon followed by the parameter name (e.g., /user/:id) to create dynamic routes that accept parameters.

  • 7. What is DebounceTime in RxJS?

    Answer: debounceTime is an RxJS operator that delays the emission of values from an observable for a specified duration, useful for reducing the frequency of events like user input.

  • 8. What is the pipe operator?

    Answer: The pipe operator in RxJS allows you to chain multiple operators together, enabling a more readable and functional approach to handling observables.

  • 9. What is the distinctUntilChanged operator?

    Answer: The distinctUntilChanged operator emits only the first value from the source observable until a new value is emitted that is different from the previous one.

  • 10. What is the filter operator in RxJS?

    Answer: The filter operator allows you to selectively emit values from an observable based on a specified condition, effectively filtering out unwanted values.
